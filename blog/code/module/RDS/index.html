Tentang RDS

RDS (Responsive Design System) adalah solusi komprehensif untuk membuat aplikasi web yang responsif dan mudah beradaptasi dengan berbagai ukuran layar. Modul ini menyediakan sistem scaling otomatis, konversi unit, dan hook untuk kustomisasi lebih lanjut.

Auto Scaling

Skala elemen UI secara otomatis berdasarkan ukuran layar pengguna dengan batasan minimum dan maksimum yang dapat dikonfigurasi.

Mode Desktop & Mobile

Dukungan konfigurasi terpisah untuk desktop dan mobile dengan breakpoint yang dapat disesuaikan.

REM Conversion

Konversi otomatis unit CSS dari px ke rem dengan dukungan untuk mode static, responsive, dan hybrid.

Lifecycle Hooks

Hook beforeUpdate dan afterUpdate untuk menambahkan logika kustom sebelum dan setelah scaling diterapkan.

Utility Functions

Berbagai fungsi utilitas untuk konversi unit, penerapan gaya, dan kontrol sistem secara terprogram.

Debug Mode

Mode debug untuk memantau perubahan skala dan mode secara real-time di konsol browser.

Dokumentasi & Cara Penggunaan

Deskripsi Umum

RDS-UE-module.js adalah modul JavaScript yang menyediakan sistem desain responsif untuk aplikasi web. Modul ini memungkinkan penyesuaian skala font dan layout secara otomatis berdasarkan ukuran viewport (desktop/mobile), dan mendukung konversi unit px ke rem secara global.

Fitur Utama

  • Skala responsif untuk desktop & mobile secara otomatis
  • Konfigurasi font-size global dan mode rem (static, responsive, hybrid)
  • Konversi stylesheet dan inline style px â†’ rem secara otomatis
  • API untuk mengubah konfigurasi, mode, dan mengaplikasikan style
  • Mendukung hook beforeUpdate dan afterUpdate untuk ekstensi
  • Debug mode untuk log detail perubahan
  • Mudah diintegrasikan ke project web modern

Cara Penggunaan

1. Import modul ke project:

import { RDS } from './rds.js';

2. Inisialisasi RDS dengan konfigurasi opsional:

RDS.init({
  breakpoint: 800,         // Custom breakpoint (default: 768)
  baseFontSize: '18px',    // Custom font size
  remMode: 'responsive',   // Pilihan: 'static' | 'responsive' | 'hybrid'
  debug: true              // Debug aktif
});

3. Terapkan style responsif pada elemen:

RDS.applyStyles(element, {
  fontSize: 24,     // Otomatis dikonversi ke rem/scaled px
  margin: '16px 8px'
});

4. Ubah mode secara manual:

RDS.setMode('desktop'); // atau 'mobile' atau 'auto'

5. Konversi seluruh CSS ke rem:

RDS.convertAllToRem();

API Lengkap

Fungsi Deskripsi
RDS.init(userConfig) Inisialisasi sistem dengan konfigurasi opsional
RDS.updateConfig(newConfig) Update konfigurasi secara dinamis
RDS.setMode(mode) Mengatur mode responsif secara manual ('auto', 'desktop', 'mobile')
RDS.destroy() Membersihkan event listener & resource RDS
RDS.getScale() Mengambil nilai skala responsif saat ini
RDS.getConfig() Mendapatkan konfigurasi aktif
RDS.getCurrentMode() Mendapatkan mode aktif (desktop/mobile)
RDS.unit(value) Konversi otomatis nilai ke format responsif (rem/scaling)
RDS.applyStyles(element, styles) Terapkan styles ke satu elemen dengan konversi otomatis
RDS.applyStylesToAll(elements, styles) Terapkan styles ke banyak elemen sekaligus
RDS.pxToRem(value, base) Konversi px ke rem (manual)
RDS.remToPx(value, base) Konversi rem ke px (manual)
RDS.responsivePx(value, base) Konversi px ke format responsif (rem + scale)
RDS.convertAllToRem(base) Konversi seluruh CSS di halaman ke rem

Konfigurasi Default

{
  desktop: {
    baseWidth: 1920,
    baseHeight: 1080,
    minScale: 0.1,
    maxScale: 1.5,
  },
  mobile: {
    baseWidth: 375,
    baseHeight: 667,
    minScale: 0.2,
    maxScale: 1.2,
  },
  baseFontSize: '16px',
  breakpoint: 768,
  forceMode: 'auto', // 'auto', 'desktop', 'mobile'
  debug: false,
  cssVarName: '--rds-scale',
  remMode: 'responsive', // 'static' | 'responsive' | 'hybrid'
  remBaseSize: 16,
  beforeUpdate: null,    // Hook sebelum update scale
  afterUpdate: null      // Hook setelah update scale
}

Penjelasan Konfigurasi

Parameter Deskripsi
desktop Konfigurasi untuk mode desktop (baseWidth, baseHeight, minScale, maxScale)
mobile Konfigurasi untuk mode mobile (baseWidth, baseHeight, minScale, maxScale)
baseFontSize Ukuran font dasar
breakpoint Lebar viewport untuk switch mode
forceMode Pilihan mode ('auto', 'desktop', 'mobile')
debug Jika true, log detail ke console
cssVarName Nama variabel CSS untuk scale
remMode Mode konversi rem ('static', 'responsive', 'hybrid')
remBaseSize Base size untuk konversi rem
beforeUpdate Hook yang dipanggil sebelum update scale
afterUpdate Hook yang dipanggil setelah update scale
Catatan Penggunaan:
  • Untuk konversi stylesheet eksternal, pastikan sheet dapat diakses (tidak cross-origin)
  • Gunakan debug: true untuk melihat log perubahan
  • Hooks dapat digunakan untuk memodifikasi perilaku scale dan mendapatkan event perubahan
  • Integrasi mudah dengan framework seperti React, Vue, atau vanilla JS
Peringatan:
  • Konversi seluruh CSS ke REM dapat mempengaruhi performa pada halaman yang sangat kompleks
  • Pastikan untuk menguji di berbagai perangkat dan browser
